# VK-common-music
Бот ищет общую музыку среди тех, у кого вы можете посмотреть аудио. Разворачивайте его в группе/паблике. В качестве метрики схожести выступает Jaro-метрика расстояния Левенштейна.
Будешь молодцом, если создашь форк и добавишь:
1)Многопоточность
2)Кэширование запросов(lru_cache?)
3)Вход не по паролю, а по токену(можешь заодно подать заявку в отдел безопасности ВК).
4)Сканирование альбомов
5)Сканирование групп(не проверял)
Репозиторий закрыт, обновления не планируются.
Для работы Longpoll попробуйте обновить его версию в настройках группы(поставьте одну из последних).
Примерное первое сообщение от группы: "Бот ищет общую музыку среди друзей.
Для этого вводите login password 0 1 2 3..
0,1,2-это числа в id пользователя(0-это ваш краткий id)"
